Print on Demand Job Manager V.3 (Python + Web App)

Upgraded the print job system from VBA to a web-based platform using Python, HTML, CSS, and JavaScript for improved flexibility and future scalability. The system reads multiple CSV files from a directory using pandas and os, logs processed files to avoid duplication, and compiles data into a DataFrame. The web interface displays the data clearly with filterable tables, styled using CSS, and retains filter states using JavaScript. The tool allows dynamic filtering, easy navigation, and consistent performance across updates. Designed for continuous improvement and actively used in real operations.
